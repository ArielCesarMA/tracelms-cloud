USER STORY: Payment Processing System

As a customer, I want to successfully pay for my order using a credit card so that I can complete my purchase.

ACCEPTANCE CRITERIA:
1. User can enter credit card details (number, expiry, CVV)
2. System validates the card information before submitting
3. Payment is processed securely through Stripe API
4. User receives order confirmation on successful payment
5. User sees clear error message on payment failure
6. Transaction is logged for auditing purposes

REQUIREMENTS:
- Support Visa, Mastercard, and American Express
- Implement PCI-DSS compliance for card handling
- Handle timeout scenarios with user-friendly messaging
- Retry failed payments up to 3 times with exponential backoff
- Display transaction ID after successful payment
